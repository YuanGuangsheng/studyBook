# 指令
* 数据绑定-插值：v-text
        <span v-text="message"></span>
* 数据绑定-插值： v-html
    * 输出真正的html代码
    * 不建议使用，容易被XSS攻击
            <div v-html="html"></div>
* 条件渲染：v-show
* 条件渲染：
    * v-if
    * v-else
    * v-else-if
* 列表渲染：v-for
* 事件处理器：v-on
* 数据绑定：v-bind
* 表单控件绑定：v-model
* v-pre
* v-clock
* v-once
